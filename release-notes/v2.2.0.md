- **Electron 30 → 42** — 지원 종료(EOL)된 Electron 30에서 현재 지원 버전으로 이동(최신 Chromium/V8, 최신 보안 패치).

- 문서상 최소 Node 버전을 **20.19 / 22.12**로 상향(소스 빌드 한정, 포터블 빌드는 영향 없음).

- **Windows**: Releases에서 v2.2.0 다운로드 → 기존 폴더에 덮어쓰기. 설정·히스토리·API 키는 그대로 유지됩니다.
- **소스 실행**: Node ≥ 20.19 / 22.12 환경에서 `npm install`을 새로 실행하세요.

## Overview

WhisperSubTranslate v2.2.0 — **minor release**. Modernizes the runtime and build toolchain (**Electron 42**), clears every dependency security advisory, and ships a refreshed offline-translation benchmark in the README. No changes to how you use the app.

## Download

- **Windows Portable (Recommended)**: `WhisperSubTranslate-v2.2.0-win-x64.zip`
  → Extract → Run `WhisperSubTranslate.exe`
- **Linux/macOS**: Run from source (see Linux Setup in README)

---

## ✨ What's New

- **Electron 30 → 42** — moves off the end-of-life Electron 30 to a currently-supported release (newer Chromium/V8, up-to-date security).
- **Clean, secure install** — `npm install` no longer prints deprecation warnings (`inflight` / `glob@7` / `boolean`), and `npm audit` reports **0 vulnerabilities** (was 18).
- **Refreshed README benchmark** — the Hy-MT2 translation-quality chart is now a **local, multilingual image** redrawn from Tencent's official Figure 1 (no remote-image dependency), highlighting the bundled models (1.8B default / 7B max).

## 🔧 Improvements

- Build toolchain bumped: `electron-builder` 24 → 26, `deepl-node` → 1.27, plus `overrides` (@electron/asar · @electron/get · temp/rimraf) to drop deprecated transitive deps.
- Documented minimum Node raised to **20.19 / 22.12** (build-from-source only; the portable build is unaffected).

## How to update

- **Windows**: download v2.2.0 from Releases and overwrite your existing folder. Settings, history, and API keys are preserved.
- **From source**: use Node ≥ 20.19 / 22.12 and run a fresh `npm install`.
