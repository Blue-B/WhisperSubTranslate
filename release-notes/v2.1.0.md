## 개요

WhisperSubTranslate v2.1.0 — **마이너 릴리즈**. 로컬 번역 엔진을 **Tencent Hy-MT2**(Apache-2.0)로 업그레이드해 품질과 언어 지원을 높이고, 저장소 구조를 정리했습니다.

## 다운로드

- **Windows 포터블(권장)**: `WhisperSubTranslate-v2.1.0-win-x64.zip`
  → 압축 해제 → `WhisperSubTranslate.exe` 실행
- **Linux/macOS**: 소스에서 직접 실행 (README의 Linux Setup 참조)

---

## ✨ 새로운 점

- **Hy-MT2 로컬 번역 엔진** — HY-MT1.5를 대체합니다. 기본 **1.8B (Q4, 약 1.13GB)**, 고품질 등급은 **7B (Q6, 약 6.16GB)**로 상향. **Apache-2.0** 라이선스에 **33개 이상 언어**를 지원하며, 동일 아키텍처라 기존 오프라인 `node-llama-cpp` 파이프라인에서 그대로 동작합니다.
- **옛 HY-MT1.5 모델 파일 자동 정리** — 이전 버전에서 남은 모델 파일을 자동으로 제거합니다.

## 🔧 개선

- 번역 프롬프트를 Hy-MT2 공식 템플릿에 맞췄고, 긴 자막 줄 처리 안정성을 높였습니다(`maxTokens` 256 → 1024).
- 저장소 구조 정리(내부): 번역 문서를 `docs/`로, 앱 아이콘을 `build/`로 이동.

## 업데이트 방법

- **Windows**: Releases에서 v2.1.0 다운로드 → 기존 폴더에 덮어쓰기. 설정·히스토리·API 키는 그대로 유지됩니다. 새 Hy-MT2 모델은 첫 로컬 번역 시 자동 다운로드되고, 옛 HY-MT1.5 파일은 자동 정리됩니다.

## Overview

WhisperSubTranslate v2.1.0 — **minor release**. Upgrades the local translation engine to **Tencent Hy-MT2** (Apache-2.0) with higher quality and broader language support, and tidies the repository layout.

## Download

- **Windows Portable (Recommended)**: `WhisperSubTranslate-v2.1.0-win-x64.zip`
  → Extract → Run `WhisperSubTranslate.exe`
- **Linux/macOS**: Run from source (see Linux Setup in README)

---

## ✨ What's New

- **Hy-MT2 local translation engine** — replaces HY-MT1.5. Default **1.8B (Q4, ~1.13 GB)**; the high-quality tier is upgraded to **7B (Q6, ~6.16 GB)**. Now **Apache-2.0** licensed with **33+ languages**, and built on the same architecture so it runs through the existing offline `node-llama-cpp` pipeline.
- **Automatic cleanup of old HY-MT1.5 model files** — leftover models from the previous version are removed automatically.

## 🔧 Improvements

- Prompt aligned to Hy-MT2's official template; long subtitle lines are handled more reliably (`maxTokens` 256 → 1024).
- Repository tidied (internal): localized docs moved into `docs/`, app icons into `build/`.

## How to update

- **Windows**: download v2.1.0 from Releases and overwrite your existing folder. Settings, history, and API keys are preserved. The new Hy-MT2 model auto-downloads on first local translation; the old HY-MT1.5 file is cleaned up automatically.
