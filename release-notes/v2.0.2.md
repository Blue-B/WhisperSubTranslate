## 개요

WhisperSubTranslate v2.0.2 — **패치 릴리즈**. 로컬(HY-MT) 번역 품질을 개선하고 Linux 호환성을 넓혔습니다. 커뮤니티 기여자 [@matbgn](https://github.com/matbgn) 님께 감사드립니다.

## 다운로드

- **Windows 포터블(권장)**: `WhisperSubTranslate-v2.0.2-win-x64.zip`
  → 압축 해제 → `WhisperSubTranslate.exe` 실행
- **Linux/macOS**: 소스에서 직접 실행 (README의 Linux Setup 참조)

---

## 🐛 버그 수정

- **로컬 번역(HY-MT) 환각·폭주 생성 수정** — 세그먼트마다 채팅 히스토리를 초기화해 문맥이 누적되지 않도록 했고, `chatWrapper: 'auto'`로 Hunyuan-MT 채팅 템플릿을 정확히 감지합니다. Tencent 권장 샘플링값(temperature 0.7, topK 20, topP 0.6, repeatPenalty 1.05)을 적용하고, 폭주 생성을 막기 위한 앱 측 안전장치로 `maxTokens: 256`을 추가했습니다.

## 🔧 개선

- **Linux 빌드 폴백** — 사전 빌드 바이너리가 없을 때 CUDA 빌드를 시도하고, CUDA를 실제로 시도한 경우에만 CPU 전용 빌드로 자동 재시도합니다 (예: RTX 5090 + nvcc 12.0처럼 미지원 GPU 아키텍처).
- **크로스플랫폼 실행기(`scripts/start.js`)** — 일부 터미널/AI 에이전트가 흘리는 `ELECTRON_RUN_AS_NODE`를 제거해 항상 GUI로 실행되도록 했습니다. Linux에서 `chrome-sandbox` setuid 비트가 없으면 SIGILL 크래시를 막기 위해 `--no-sandbox`를 주입하며, 이때 보안이 약화됨을 알리는 경고를 출력합니다.

## 업데이트 방법

- **Windows**: Releases에서 v2.0.2 다운로드 → 기존 설치 폴더에 덮어쓰기. 설정·히스토리·API 키는 그대로 유지됩니다.

## Overview

WhisperSubTranslate v2.0.2 — **patch release**. Improves local (HY-MT) translation quality and broadens Linux compatibility. Thanks to community contributor [@matbgn](https://github.com/matbgn).

## Download

- **Windows Portable (Recommended)**: `WhisperSubTranslate-v2.0.2-win-x64.zip`
  → Extract → Run `WhisperSubTranslate.exe`
- **Linux/macOS**: Run from source (see Linux Setup in README)

---

## 🐛 Fixes

- **Local translation (HY-MT) hallucinations & runaway generation** — chat history is now reset before every segment so context no longer accumulates, and `chatWrapper: 'auto'` lets Hunyuan-MT pick the correct chat template. Adopts Tencent's recommended sampling (temperature 0.7, topK 20, topP 0.6, repeatPenalty 1.05) and adds `maxTokens: 256` as an app-side safety cap to curb runaway output.

## 🔧 Improvements

- **Linux build fallback** — when no prebuilt binary is available, the installer attempts a CUDA build and only retries a CPU-only build if a CUDA build was actually attempted (e.g. unsupported GPU architectures like RTX 5090 + nvcc 12.0).
- **Cross-platform launcher (`scripts/start.js`)** — unsets a leaked `ELECTRON_RUN_AS_NODE` so the app always starts in GUI mode. On Linux, when `chrome-sandbox` lacks the setuid bit it injects `--no-sandbox` to avoid SIGILL crashes, printing a warning that the sandbox is reduced.

## How to update

- **Windows**: download v2.0.2 from Releases and overwrite your existing folder. Settings, history, and API keys are preserved.

WhisperSubTranslate v2.0.2 — **wydanie poprawkowe**. Poprawia jakość lokalnego tłumaczenia (HY-MT) i rozszerza zgodność z systemem Linux. Podziękowania dla współtwórcy społeczności [@matbgn](https://github.com/matbgn).

## 🐛 Poprawki

- **Halucynacje i niekontrolowane generowanie w tłumaczeniu lokalnym (HY-MT)** — historia czatu jest teraz resetowana przed każdym segmentem, więc kontekst się nie kumuluje, a `chatWrapper: 'auto'` pozwala Hunyuan-MT dobrać właściwy szablon czatu. Przyjęto zalecane przez Tencent próbkowanie (temperature 0.7, topK 20, topP 0.6, repeatPenalty 1.05) i dodano `maxTokens: 256` jako limit bezpieczeństwa po stronie aplikacji, ograniczający niekontrolowane wyjście.

## 🔧 Ulepszenia

- **Awaryjny build dla Linux** — gdy brak gotowego binarium, instalator próbuje budowy z CUDA i ponawia build wyłącznie dla CPU tylko wtedy, gdy build CUDA był faktycznie podjęty (np. nieobsługiwane architektury GPU jak RTX 5090 + nvcc 12.0).

W systemie Linux, gdy `chrome-sandbox` nie ma bitu setuid, dodaje `--no-sandbox`, aby uniknąć awarii SIGILL, wypisując ostrzeżenie o osłabionym sandboxie.

- **Windows**: pobierz v2.0.2 z Releases i nadpisz istniejący folder. Ustawienia, historia i klucze API są zachowane.
