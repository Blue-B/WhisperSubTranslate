Privacy Policy

Last updated: December 2024

Key Summary:

WhisperSubTranslate does not collect user data. Speech recognition runs locally on your computer. Translation (optional) sends subtitle text to external APIs.

1. Information We Collect

WhisperSubTranslate desktop application does NOT collect:

  • Personal identification information (name, email, phone, etc.)
  • Video/audio files you process
  • Generated subtitle content
  • Usage patterns or analytics data
  • IP addresses or location information

2. How Data is Processed

WhisperSubTranslate performs subtitle extraction locally on your device:

  • Local Processing: All speech recognition and subtitle generation is performed directly on your computer.
  • File Storage: Generated subtitle files are only saved to locations you specify.
  • No Servers: No data is transmitted to external servers.

3. Translation Feature

When using the translation feature, extracted subtitle text may be sent to translation API providers:

  • DeepL: DeepL Privacy Policy
  • MyMemory: MyMemory Service
  • GPT-5-nano (OpenAI): OpenAI Privacy Policy

Translation API usage is optional and requires you to enter your own API key.

4. Website Visits

When visiting this website:

  • GitHub Pages: Hosted on GitHub's servers, subject to GitHub's privacy policy.
  • Cookies: Only minimal cookies required for essential functionality.
  • Analytics: Google Analytics may be used; only anonymized data is collected.

5. Third-Party Services

The application uses the following open-source technologies:

  • whisper.cpp - C++ implementation of OpenAI Whisper
  • Electron - Desktop application framework
  • ffmpeg - Audio/video processing

All these components run locally on your computer.

6. Children's Privacy

WhisperSubTranslate is not directed at children under 13. We do not knowingly collect personal information from children under 13.

7. Changes

This privacy policy may be updated as needed. Significant changes will be announced on this page.
