Features

Everything you need for subtitle work

Professional-grade transcription and translation without the complexity.

Complete Privacy

Speech recognition runs locally. Your files never leave your computer. Translation (optional) uses online APIs.

GPU Acceleration

CUDA support for NVIDIA GPUs. Process videos 5-10x faster than CPU-only.

Multi-language Translation

Free translation with MyMemory (no API key). DeepL/GPT-5-nano available with your own API keys.

Batch Processing
